以下の手順に従い、device-inventory の単体テストコードを作成してください。

## 入力ファイル（必ず読み込むこと）

1. UI設計書: docs/03-features/flask-app/device-inventory/ui-specification.md
2. 機能設計書: docs/03-features/flask-app/device-inventory/workflow-specification.md
3. 単体テスト観点表: docs/05-testing/unit-test/unit-test-perspectives.md
4. 単体テスト実装ガイド: docs/05-testing/unit-test/unit-test-guide.md
5. テストコードレビュー用プロンプト: docs/05-testing/unit-test/テストコードレビュー用プロンプト.md

※ 上記パスにファイルがない場合は、サブフォルダ（書面レビュー対象資料/ 等）内を探索すること。

## 作業手順

1. UI設計書と機能設計書を読み、機能の振る舞いを把握する
2. 単体テスト観点表を読み、該当する機能に必要な観点を選択する
3. 単体テスト実装ガイドの「テストコード実装例」を参照し、コードの書き方を確認する
4. 設計書に記載された具体的な値（項目名、バリデーション条件、エラーメッセージ等）をテストコードに反映する
5. 設計書の実装例に記載された単体テスト対象の内容をすべて漏らさずテストコードを生成する
6. 各テストコードが単体テスト観点表のうち、どの観点に該当するかコメントで各テストコードに記載する
7. 各テストコードの実行する内容、想定結果をコメントで各テストコードに記載する
8. テストコードレビュー用プロンプトのチェックリストに従いセルフレビューを実施し、指摘がある場合はテストコードとテスト項目書を修正する

## 出力先

`tests/unit/` 配下は `src/iot_app/` のレイヤー構成を1:1でミラーする。
対象機能が属するレイヤーに応じて以下のパスに出力すること。

| レイヤー  | 出力パス                                       | 例                                   |
| --------- | ---------------------------------------------- | ------------------------------------ |
| Service層 | `tests/unit/services/test_device-inventory_service.py` | `test_customer_dashboard_service.py` |
| Model層   | `tests/unit/models/test_device-inventory.py`           | `test_dashboard.py`                  |
| form層   | `tests/unit/forms/test_device-inventory.py`           | `test_dashboard.py`                  |

複数レイヤーにまたがる場合は、レイヤーごとに別ファイルとして出力する。
例: Service層とModel層の両方をテストする場合 → `services/` と `models/` に1ファイルずつ作成。
原則上記の三点が必要になるが、その他実装が必要ファイルが存在する場合は合わせて実装を行うこと（対象ファイルは設計書上から解析する）

## 出力形式

pytestのテストコードとして出力すること。以下のルールに従う：
- クラスのdocstringに、対応する観点表のセクション番号を記載する
- 各テストメソッドのdocstringに、対応する観点表の項目番号と概要を記載する
- AAAパターン（Arrange-Act-Assert）に従う
- @pytest.mark.unit マーカーを付与する
- 外部依存（DB、API等）はすべてモック化する
- 具体的なコードパターンは単体テスト実装ガイドの「テストコード実装例」を参照する

## 注意事項

- 観点表のすべての項目を機械的に適用しないこと。設計書に該当する機能・項目がある観点のみ選択する
- 設計書に明記されていない挙動は、テストコード内にコメントで「# TODO: 設計書に記載なし、要確認」と記載する
- 原則としてテスト駆動開発で実装を行うため、未実装のテスト対象であってもテストコードを生成する
