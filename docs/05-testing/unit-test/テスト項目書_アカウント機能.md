## 適用外の観点（設計書に該当機能なし）

| 観点                        | 理由                                           |
| --------------------------- | ---------------------------------------------- |
| 1.4 ログ出力機能            | ログはミドルウェア・共通モジュールが担当        |
| 3.2 登録機能                | アカウント機能に新規登録処理なし               |
| 3.4 削除機能                | アカウント機能に削除処理なし                   |
| 3.5 CSV エクスポート機能    | アカウント機能に CSV 出力処理なし              |
| 1.1.3 数値範囲チェック      | 数値入力項目なし（`user_id` は内部IDのため対象外） |
| 1.1.4 日付形式チェック      | 日付入力項目なし                               |
| 1.1.5 メールアドレス形式    | メールアドレスは読み取り専用（入力なし）       |
