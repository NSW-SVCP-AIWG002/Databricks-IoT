# シナリオ試験項目書 - 分析機能

## 概要

| 項目     | 内容                                                       |
| -------- | ---------------------------------------------------------- |
| 試験種別 | シナリオ試験（E2Eテスト）                                  |
| 対象機能 | 業種別ダッシュボード / 顧客作成ダッシュボード / AIチャット |
| 試験方式 | 手動（ブラウザ操作）                                       |

## テスト環境・前提条件

- ブラウザ: Chrome 最新版
- 画面解像度: 1920×1080
- ログイン済みユーザーが存在すること
- 組織・デバイス・センサーデータが DB / Unity Catalog に存在すること

## シナリオ一覧

| シナリオID | 対象機能                            | タイトル                                             |
| ---------- | ----------------------------------- | ---------------------------------------------------- |
| SC-NAV-001 | グローバルナビゲーション            | サイドバー展開・各分析画面への遷移                   |
| SC-IDS-001 | 業種別ダッシュボード                | 店舗モニタリングからデバイス詳細・CSV取得            |
| SC-IDS-002 | 業種別ダッシュボード                | デバイス詳細での期間指定・グラフ操作                 |
