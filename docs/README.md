## 📝 ドキュメント作成ガイドライン

新しいドキュメントを作成する際は、以下のガイドラインに従ってください：

1. **適切なディレクトリに配置する**
   - 機能仕様は `03-features/` へ
   - API仕様は `05-api/` へ
   - アーキテクチャの決定は `01-architecture/decisions/` へ

2. **テンプレートを活用する**
   - `08-templates/` にある各種テンプレートを使用

3. **Markdownフォーマットで記述する**
   - 見出しは階層的に構成
   - コードブロックには言語を指定
   - 図表にはMermaidを活用

4. **バージョン管理**
   - 重要な変更は履歴を残す
   - 日付と変更者を明記
