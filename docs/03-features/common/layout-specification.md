## 基本情報

| 項目             | 内容                                                                                                |
| ---------------- | --------------------------------------------------------------------------------------------------- |
| コンポーネント名 | グローバルメニュー（左サイドバー）                                                                  |
| 適用範囲         | 全画面共通                                                                                          |
| 実装ファイル     | `src/iot_app/templates/base.html`                                                                   |
| 関連CSSファイル  | `src/iot_app/static/css/components/sidebar.css`（新規作成・`main.css` に `@import` を追加すること） |
| CSS変数定義      | `src/iot_app/static/css/variables.css` に以下を追加する                                             |
| 関連JSファイル   | `src/iot_app/static/js/main.js`（`openImportModal()` 等の共通関数・サイドバークリック開閉ハンドラを定義） |

**variables.css に追加するCSS変数:**

```css
/* variables.css の :root {} 内に追加 */
:root {
  --nav-bkg-color-a: #2A3847;   /* ノンアクティブ */
  --nav-bkg-color-b: #2E3D4D;   /* ホバー中 */
  --nav-bkg-color-c: #596775;   /* 現在地 */
}
```

### Jinja2テンプレートブロック

子テンプレートは `{% extends 'base.html' %}` で継承し、以下のブロックを上書きする。

| ブロック名              | 用途                                    | 必須 |
| ----------------------- | --------------------------------------- | ---- |
| `{% block title %}`     | ブラウザタブのタイトル（`<title>`タグ） | ○    |
| `{% block content %}`   | メインコンテンツエリア                  | ○    |
| `{% block extra_css %}` | ページ固有CSSの追加読み込み             | -    |
| `{% block extra_js %}`  | ページ固有JSの追加読み込み              | -    |

## 色・状態仕様

### 背景色

| 状態           | 記号 | 対象                                                                   | カラーコード |
| -------------- | ---- | ---------------------------------------------------------------------- | ------------ |
| ノンアクティブ | A    | 閉じている親項目 / 展開中の親配下にある非現在地の子項目                | `#2A3847`    |
| ホバー中       | B    | クリックで展開中の親ヘッダー行・ホバー中の子メニュー項目               | `#2E3D4D`    |
| 現在地         | C    | 現在開いているページに対応する**子項目のみ**                           | `#596775`    |

### テキスト色

| 対象                           | 状態             | カラーコード |
| ------------------------------ | ---------------- | ------------ |
| 親メニュー項目・直接リンク項目 | 通常・ホバー共通 | `#ecf0f1`    |
| 子メニュー項目                 | 通常             | `#bdc3c7`    |
| 子メニュー項目                 | ホバー中         | `#ecf0f1`    |
| 子メニュー項目                 | 現在地           | `#ecf0f1`    |

### 状態遷移のまとめ

```
親項目をクリック         → サブメニューを展開（他の展開中サブメニューは自動で閉じる）
展開中の親項目をクリック → サブメニューを閉じる

親項目にカーソルを乗せる → 親がB色に変化（展開・遷移はしない）
親項目からカーソルを外す → 親がA色に戻る

ホバー中の子項目 → B色（カーソルを外すと解除）
現在地の子項目 → C色（request.endpoint完全一致・ホバー有無に関わらず維持）
その他の子 → A色
```

## インタラクション仕様

### 項目種別ごとの動作

| 対象                         | ホバー                                                | クリック                                                      |
| ---------------------------- | ----------------------------------------------------- | ------------------------------------------------------------- |
| サイドバー全体               | 幅が60px→240pxに拡張（アイコンのみ→フルメニュー表示） | -                                                             |
| 親メニュー項目               | 背景色をB色に変化（遷移なし）                         | サブメニューをトグル開閉（アコーディオン：一度に1つのみ展開） |
| 子メニュー項目               | 背景色をB色に変化（ハイライト）                       | 対応URLへ遷移                                                 |
| 直接リンク項目（ホーム）     | ハイライト（カーソル変化）                            | `/analysis/industry-dashboard/store-monitoring` へ遷移        |
| 直接リンク項目（インポート） | ハイライト（カーソル変化）                            | CSVインポートモーダルを表示（モーダルID: `csv-import-modal`） |

### アクティブ状態の維持

- ページロード時の初期状態は全メニュー閉じ（サブメニューの自動展開は行わない）
- 現在地の子項目はC色を維持（`request.endpoint` の完全一致で判定。ホバー状態に関わらず常に適用）

### 実装方針

- **現在地判定（サーバーサイド）**: `request.endpoint` の完全一致で現在地の子項目（C色）を決定する
- **サブメニュー展開・開閉（クライアントサイド）**: 親メニュー項目クリックで `.is-open` クラスをトグル（JavaScript制御）。アコーディオン動作（一度に1つのみ展開）

**C色判定のJinja2実装例:**

```html
<a href="{{ url_for('admin.user_list') }}"
   class="nav__item--child {% if request.endpoint == 'admin.user_list' %}nav__item--active{% endif %}">
  ユーザー管理
</a>
```

`nav__item--active` モディファイアに `background-color: var(--nav-bkg-color-c)` を適用する。

**表示制御の実装方針:**
- `-` の項目はHTML上でレンダリングしない（Jinja2のロール条件分岐で制御）

- ロールの判定には `g.current_user_type_id` を使用（`before_request` でセット済み）
- ユーザータイプIDは定数で管理する（共通設計参照）

**要素構成図:**

```
┌──────────────────────────────┐
│  [i]  管理機能          ▲    │  ← 展開中（クリック済み）
```

## 実装例

### HTMLレイアウト構造

```html
<nav class="navigation">
  <ul class="nav__menu">  <!-- flex: 1 で残余スペースを占有。overflow-y: auto でスクロール可 -->

    <!-- 直接リンク項目（href遷移） -->
    <li class="nav__item--direct">
      <a class="nav__link" href="{{ url_for('analysis.store_monitoring') }}">
        <span class="nav__icon"><img src="..."></span>
        <span class="nav__link-text">ホーム</span>
      </a>
    </li>

    <!-- 直接リンク項目（モーダル表示） -->
    <li class="nav__item--direct">
      <div class="nav__link" onclick="openImportModal()">
        <span class="nav__icon"><img src="..."></span>
        <span class="nav__link-text">インポート</span>
      </div>
    </li>

    <!-- 親メニュー項目（サブメニューあり） -->
    <li class="nav__item--parent">
      <div class="nav__link">
        <span class="nav__icon"><img src="..."></span>
        <span class="nav__link-text">管理機能</span>
        <span class="nav__arrow">▼</span>
      </div>
      <div class="nav__submenu">
        <a class="nav__item--child {% if request.endpoint == 'admin.user_list' %}nav__item--active{% endif %}"
           href="{{ url_for('admin.user_list') }}">ユーザー管理</a>
        ...
      </div>
    </li>

  </ul>
  <ul class="nav__account">  <!-- nav__menu が flex: 1 を占有するため自動的に最下部に配置 -->
    <!-- アカウント：nav__item--parent と同構造 -->
  </ul>
</nav>

<!-- メインコンテンツ（navigation の兄弟要素として配置） -->
<main class="main-content">
  {% block content %}{% endblock %}
</main>
```

### CSS（sidebar.css）

```css
/* (1) グローバルメニュー */
```

### JavaScript（main.js）

```javascript
/* サイドバーのサブメニュークリック開閉（アコーディオン） */
document.querySelectorAll('.nav__item--parent > .nav__link').forEach(function(link) {
  link.addEventListener('click', function() {
    var parent = this.closest('.nav__item--parent');
    var isOpen = parent.classList.contains('is-open');

    // 他の開いているサブメニューを閉じる（アコーディオン）
    document.querySelectorAll('.nav__item--parent.is-open').forEach(function(el) {
      el.classList.remove('is-open');
    });

    // クリックした項目をトグル
    if (!isOpen) {
      parent.classList.add('is-open');
    }
  });
});
```
