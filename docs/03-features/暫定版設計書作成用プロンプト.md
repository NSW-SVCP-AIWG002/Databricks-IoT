**目的**：
以下の「Input要件資料」を入力として、「output用ガイド資料」に基づき、対話型AIチャットの設計資料（UI仕様書、ワークフロー仕様書）を生成してください。

**Input要件資料**:
- `02-requirements\functional-requirements.md`：機能要件、重要度高
- `02-requirements\non-functional-requirements.md`：非機能要件
- `02-requirements\technical-requirements.md`：技術要件
- `03-features\common\unity-catalog-database-specification.md`：UnityCatalogデータベース設計書
- `03-features\common\app-database-specification.md`：アプリケーションデータベース設計書

**Output用ガイド資料**:
- `03-features\README.md`：ディレクトリ概要
- `03-features\feature-guide.md`：機能仕様書作成用ガイド資料
- `03-features\templates\ui-specification-template.md`：UI仕様書テンプレート
- `03-features\templates\workflow-specification-template.md`：ワークフロー仕様書テンプレート

**タスク**：
1. **完了したタスクを管理するためのタスクリストを生成し、順次記帳する**
2. **02 配下の要件資料を読み込み、要件を完全に理解して要約する**
3. **03/README.md、03/feature-guide.md  のプロジェクト方針・規約に沿った構成で資料を作成する**
4. **03/templates 配下のテンプレート構造を厳守し、それを埋める形で設計資料を生成する**
5. **テンプレートに無い要素が必要な場合は、適切な章を新設し、その理由を説明する**
6. **資料生成後は02 配下の要件資料と突合し、要件漏れがないかを確認する**

**出力形式**：
- `flask-app\conversational-ai-chat\tmp\design_tasklist.md`:タスクリストを出力、完了次第順次記帳
- `flask-app\conversational-ai-chat\tmp\design_preparation.md`:「把握した要件」「解析した不足情報」、「新設した章の詳細」を出力
- `flask-app\conversational-ai-chat\README.md `:READMEを出力
- `flask-app\conversational-ai-chat\ui-specification.md`:UI仕様書を出力
- `flask-app\conversational-ai-chat\workflow-specification.mdd`：ワークフロー仕様書を出力
