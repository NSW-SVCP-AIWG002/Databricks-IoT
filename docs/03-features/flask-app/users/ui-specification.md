## 基本情報

| 項目         | 内容                                               |
| ------------ | -------------------------------------------------- |
| 画面ID       | `ADM-005`                                          |
| 画面名       | ユーザー管理画面                                   |
| 画面パス     | `/admin/users`                                     |
| アクセス権限 | システム保守者、管理者、販社ユーザ、サービス利用者 |
| 親画面       | グローバルメニュー（管理機能）                     |
| 子画面       | なし                                               |

**注:**
- データスコープ制限（全ユーザー共通）:
  - `organization_closure` テーブルで下位組織リストを取得
  - そのリストに該当する `organization_id` のユーザーのみ表示

- 画面サイズは1920×1080を基準に設計
- 画面左部にグローバルメニューカラムを表示

- 登録、更新、参照画面は同画面内でモーダルとして表示
- ページネーション形式: 「1 … n-1 n n+1 … 最大ページ」
