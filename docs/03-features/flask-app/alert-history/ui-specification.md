**2-1: 期間（開始）検索**
- 概要: アラート発生日時の検索範囲の開始日時を指定
- ラベル: 「期間（開始）」
- 入力形式: YYYY/MM/DD HH:mm
- 初期値: 現在日時から7日前の00:00
- カレンダーアイコンクリックでDateTimePickerを表示

**2-2: 期間（終了）検索**
- 概要: アラート発生日時の検索範囲の終了日時を指定
- ラベル: 「期間（終了）」
- 入力形式: YYYY/MM/DD HH:mm
- 初期値: 現在日時の23:59
- カレンダーアイコンクリックでDateTimePickerを表示

**2-3: デバイス名検索**
- 概要: デバイス名で部分一致検索
- ラベル: 「デバイス名」
- プレースホルダー: なし
- 入力制限: 入力可能文字列長は最大100文字

**2-4: 設置場所検索**
- 概要: 設置場所で部分一致検索
- ラベル: 「設置場所」
- プレースホルダー: なし
- 入力制限: 入力可能文字列長は最大100文字

**2-5: アラート名**
- 概要: アラート名で部分一致検索
- ラベル: 「アラート名」
- プレースホルダー: なし
- 入力制限: 入力可能文字列長は最大100文字

**2-6: アラートレベル**
- 概要: アラートレベルでフィルタリング
- ラベル: 「アラートレベル」
- 選択肢:
  - すべて（デフォルト）
  - DB: alert_level_masterから全件取得
    - Critical（重大）
    - Warning（警告）
    - Info（情報）
- 選択肢のソート順
  - 「すべて」が最上位
  - 以降はalert_level_master.alert_level_idの昇順

**2-7: ステータス**
- 概要: アラートステータスでフィルタリング
- ラベル: 「ステータス」
- 選択肢:
  - すべて（デフォルト）
  - DB: alert_status_masterから全件取得
    - 発生中
    - 復旧済み
- 選択肢のソート順
  - 「すべて」が最上位
  - 以降はalert_status_master.alert_status_idの昇順

**2-8: ソート項目**
- 概要： 一覧の全体ソート項目を選択
- ラベル:「ソート項目」
- プレースホルダー：なし
- 選択肢：
  - DB：sort_item_masterテーブルから動的取得
- 選択肢のソート順
  - sort_item_master.sort_orderの昇順
- デフォルト: アラート発生日時

**2-9: ソート順**
- 概要：一覧の全体ソート順序を選択
- ラベル：「ソート順」
- プレースホルダー：なし
- 選択肢：
  - 昇順
  - 降順
- デフォルト: 降順

| 5-5 | アラートレベル | 15% |

**テーブル設定:**
- テーブル全体の最小幅: 1200px
- 1ページあたりの表示件数: 25件（固定）
- デフォルトソート: アラート発生日時（降順）- 最新のアラートが上に表示
- 空の場合の表示: 空のテーブル

- スタイル: ラベル

**5-3: 設置場所カラム**

- スタイル: ラベル

**5-4: アラート名カラム**
- 概要: アラート設定の名称を表示
- スタイル: ラベル

**5-5: アラートレベルカラム**
- 概要: アラートのレベルを表示
- スタイル: バッチ

  - Critical: 赤
  - Warning: 黄

**5-6: ステータスカラム**
