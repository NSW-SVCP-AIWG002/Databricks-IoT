## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | `ADM-009` |
| 画面名 | 組織管理画面（組織一覧・検索） |
| 画面パス | `/admin/organizations` |
| アクセス権限 | システム保守者、管理者、販社ユーザ |
| 親画面 | グローバルメニュー（管理機能） |
| 子画面 | [組織参照画面（ADM-012）](#6-組織参照モーダル)、[組織登録画面（ADM-010）](#7-組織登録モーダル)、[組織更新画面（ADM-011）](#8-組織更新モーダル) |

**注:**
- 画面IDは、機能要件定義書の画面一覧と対応
- 自社に紐づく組織のみアクセス可能（データスコープ制限）
- サービス利用者は組織管理画面にアクセス不可（グローバルメニューで非表示）

### (2) アクション

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────┐
│（2）アクション: (2.1) [エクスポート] (2.2) [登録] (2.3) [削除] │
└─────────────────────────────────────────────────────────────┘
```

**2-1: エクスポートボタン**
- 概要: 組織情報のCSVエクスポートを行うためのアクションボタン
- クリック時: 組織情報のCSVエクスポート処理を実行
- 出力範囲: データテーブルに表示されている全ページの値

**2-2: 登録ボタン**
- 概要: 組織情報の登録を行うためのアクションボタン
- クリック時: 組織登録モーダル（7）を表示

**2-3: 削除ボタン**
- 概要: 組織情報の削除を行うためのアクションボタン
- クリック時: 削除確認モーダル（9）を表示
- 活性条件: データテーブルのいずれかの選択チェックボックスがチェック済みの場合
- 非活性条件: データテーブルのすべての選択チェックボックスが未チェックの場合

**3-1: 折りたたみボタン**
- 概要: 検索フォームを折りたたむためのボタン
- クリック時: 検索フォームを折りたたむ

**3-2: 組織名入力**
- 概要: 組織名で部分一致検索を行う
- ラベル: 「組織名」

**3-3: 組織種別ドロップダウン**
- 概要: 組織の種別でフィルタリングするための選択項目。単一選択
- ラベル: 「組織種別」
- 選択肢:
  - すべて

## 備考

### 画面固有の仕様

- 組織管理画面は、デバイス管理・ユーザー管理と同様の標準的なCRUD画面構成
- 登録・更新・参照画面はモーダル形式で表示
- 削除は一覧画面でチェックボックス選択後、確認モーダルを表示して実行

### データスコープ制限

- 自社に紐づく組織のみアクセス可能
- 一覧・検索・詳細・更新・削除・CSVエクスポートすべての操作で組織閉包テーブル(organization_closure)によるフィルタリングが適用される
- アクセス権限外の組織IDを指定した場合は404エラー（「データが見つかりません」）

### Databricks連携

- 組織登録時: Databricks ワークスペースグループを作成
- 組織更新時: Databricks ワークスペースグループを更新
- 組織削除時: Databricks ワークスペースグループを削除
- 連携処理の詳細は [ワークフロー仕様書](./workflow-specification.md) を参照
