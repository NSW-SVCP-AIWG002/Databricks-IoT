## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | `ADM-009`, `ADM-010`, `ADM-011`, `ADM-012` |
| 画面名 | 組織管理画面 |
| 画面パス | `/admin/organizations` |
| アクセス権限 | システム保守者、管理者、販社ユーザー |
| 親画面 | グローバルメニュー（管理機能） |
| 子画面 | なし |

**注:** サービス利用者は組織管理画面にアクセス不可（グローバルメニューで非表示）

**2-1: 組織名検索**
