## 要素詳細

### (2) 検索フォーム

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────┐
│ (2) 検索フォーム                                                     │
├─────────────────────────────────────────────────────────────────────┤
│ (2.1) メール種別          (2.2) 送信先メールアドレス                │
│ ┌────────────────────┐   ┌────────────────────┐                    │
│ │▼ 選択してください  │   │                    │                    │
│ └────────────────────┘   └────────────────────┘                    │
│                                                                     │
│ (2.3) キーワード（件名・本文）                                      │
│ ┌──────────────────────────────────────────────────────────────┐  │
│ │                                                              │  │
│ └──────────────────────────────────────────────────────────────┘  │
│                                                                     │
│ (2.4) 送信日時（開始）    (2.5) 送信日時（終了）                    │
│ ┌────────────────────┐   ┌────────────────────┐                    │
│ │YYYY/MM/DD          │   │YYYY/MM/DD          │                    │
│ └────────────────────┘   └────────────────────┘                    │
│                                                                     │
│ (2.6) [検索]  (2.7) [クリア]                                       │
└─────────────────────────────────────────────────────────────────────┘
```

**2-1: メール種別ドロップダウン**
- 概要: メール送信履歴をメール種別でフィルタリングするための選択項目。複数選択可能
- ラベル: 「メール種別」
- 選択肢:
  - アラート通知
  - 招待メール
  - パスワードリセット
  - システム通知

**2-2: 送信先メールアドレス入力**
- 概要: 送信先のメールアドレスでフィルタリングするための入力項目。部分一致検索
- ラベル: 「送信先メールアドレス」
- プレースホルダー: 「user@example.com」

**2-3: キーワード入力**
- 概要: メール件名または本文に含まれる文字列でフィルタリングするための入力項目。部分一致検索
- ラベル: 「キーワード（件名・本文）」
- プレースホルダー: 「検索キーワードを入力」

**2-4: 送信日時（開始）**
- 概要: 指定日時以降のメール送信履歴を取得するための日付選択項目
- ラベル: 「送信日時（開始）」
- 入力タイプ: date
- プレースホルダー: 「YYYY/MM/DD」

**2-5: 送信日時（終了）**
- 概要: 指定日時以前のメール送信履歴を取得するための日付選択項目
- ラベル: 「送信日時（終了）」
- 入力タイプ: date
- プレースホルダー: 「YYYY/MM/DD」

**バリデーション:**

| 項目 | バリデーションルール | エラーメッセージ |
|------|---------------------|-----------------|
| 2-2 送信先メールアドレス | 最大255文字 | 「送信先メールアドレスは255文字以内で入力してください」 |
| 2-2 送信先メールアドレス | メールアドレス形式（入力された場合） | 「有効なメールアドレスを入力してください」 |
| 2-3 キーワード | 最大255文字 | 「キーワードは255文字以内で入力してください」 |
| 2-4, 2-5 送信日時 | 開始日時 ≦ 終了日時 | 「開始日時は終了日時以前の日付を指定してください」 |

```

│種別 │        │           │              │          │            │
├────┼────────┼───────────┼──────────────┼──────────┼────────────┤

├────┼────────┼───────────┼──────────────┼──────────┼────────────┤

└────┴────────┴───────────┴──────────────┴──────────┴────────────┘
                 (3.7) [< 1 2 3 ... 10 >] 全250件
```

**カラム幅:**

| No | カラム名 | 幅 |
|----|---------|-----|
| 3-1 | メール種別 | 12% |
| 3-2 | 送信者 | 18% |
| 3-3 | 件名 | 25% |
| 3-4 | 本文（抜粋） | 30% |
| 3-5 | 送信日時 | 12% |
| 3-6 | アクション | 8% |

**テーブル設定:**
- テーブル全体の最小幅: 1200px（この幅を下回る場合は横スクロール）
- 1ページあたりの表示件数: 25件（固定）
- 最大表示件数: 1000件
- デフォルトソート: 送信日時（降順）
- 空の場合の表示: 「メール通知履歴が見つかりませんでした」
- ローディング中の表示: スケルトンスクリーン

**3-1: メール種別カラム**
- 概要: メールの種別をバッジ形式で表示
- スタイル:
  - アラート通知: 赤色バッジ
  - 招待メール: 青色バッジ
  - パスワードリセット: 黄色バッジ
  - システム通知: 緑色バッジ

**3-3: 件名カラム**
- 概要: メールの件名を表示。一覧では省略表示（最大50文字）
- 51文字以上の場合: 「...」で省略

**3-4: 本文（抜粋）カラム**
- 概要: メールの本文を表示。一覧では省略表示（最大100文字）
- 101文字以上の場合: 「...」で省略
- 改行は削除してスペースに置換

**3-5: 送信日時カラム**
- 概要: メール送信日時を表示
- 表示形式: `YYYY/MM/DD HH:mm`

**3-6: アクションボタンカラム**
- 概要: メール通知履歴の詳細を表示するボタン
- ボタン: 「詳細」
- クリック時: メール通知履歴参照モーダル（5）を開く

**ソートインジケーター:**
- デフォルト: ↕（ソート可能を示す）
- 昇順: ↑
- 降順: ↓

## 備考

### 特記事項

- メール通知履歴は参照専用機能のため、登録・更新・削除ボタンは配置しません
- データテーブルにチェックボックスは配置しません
- デフォルトソートは送信日時（降順）で、最新のメールが上部に表示されます
