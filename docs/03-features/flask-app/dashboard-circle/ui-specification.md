
**1: ガジェット名**
- 概要: ガジェット名を表示

**1.1: 設定アイコン（⚙）**

- メニュー項目:

**3: 円グラフ**
- 概要: センサーデータの割合を円グラフで表示
- 描画方式: Apache ECharts
- 表示形式: 各項目の値を割合として円グラフで表示
- セグメントの色: EChartsデフォルトカラーパレット
- ホバー時: 項目名と値をツールチップ表示

**4: 凡例**
- 概要: 表示中の項目名と色を表示
- 表示形式: ■ {項目名}
- 配置: グラフ下部中央

## 備考

### 棒グラフ機能との差異

円グラフ機能は棒グラフ機能と比較して、以下の要素が不要：
- 表示時間単位選択（時/日/週/月）
- 集計時間幅選択（1/2/3/5/10/15分）
- 時間帯日時ピッカー
- 更新ボタン
- CSV出力ボタン

### データ取得タイミング

- 画面初期表示時にシルバー層から最新データを取得
- ガジェット登録時に選択した項目のデータのみ表示
