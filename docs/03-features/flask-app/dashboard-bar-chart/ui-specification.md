### (2) グループ名 / (3) CSV出力ボタン

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (2) グループ名                                                            (3) [CSV出力]  │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**2: グループ名**
- 概要: 現在表示中のグループ名を表示
- 配置: 左寄せ

**3: CSV出力ボタン**
- 概要: 現在表示中のグラフデータをCSVファイルとしてダウンロード
- 文言: 「CSV出力」
- スタイル: 青背景、白文字
- 配置: 右寄せ
- 出力ファイル名: `bar_chart_{device_id}_{yyyyMMddHHmmss}.csv`

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐

│                                                                                         │
│                            (5) ■ 凡例 (単位)                                             │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**4: 棒グラフ**
- 概要: センサーデータを棒グラフで時系列表示
- 描画方式: Apache ECharts
- X軸: 時間軸（表示時間単位に応じた目盛り）
- Y軸: 値（左側に表示、選択項目の単位）
- バーの色: #2272B4（--color-primary）
- 最大表示件数: 100バー

**5: 凡例**
- 概要: 表示中の項目名と単位を表示
- 表示形式: ■ {項目名}（{単位}）
- 配置: グラフ下部中央
- 例: 「■ 通信通電時間（m）」

### (6) 表示時間単位

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (6) 表示時間単位： [(6.1) 時] [(6.2) 日] [(6.3) 週] [(6.4) 月]                          │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**6: 表示時間単位ラベル**
- 概要: 表示時間単位の選択ボタングループ
- ラベル: 「表示時間単位：」

**6.1: 時ボタン**
- 概要: 時単位でデータを表示
- 文言: 「時」
- データソース: シルバー層（silver_sensor_data）
- スタイル: 選択時は青背景・白文字

**6.2: 日ボタン**
- 概要: 日単位でデータを表示
- 文言: 「日」
- データソース: **TODO**（新規時間単位サマリーテーブル）
- 集計間隔: 1時間
- 表示目盛り: 2時間ごと（0時, 2時, ..., 22時）
- データ本数: 24本
- スタイル: 選択時は青背景・白文字

**6.3: 週ボタン**
- 概要: 週単位でデータを表示
- 文言: 「週」
- データソース: ゴールド層（daily_summary）
- 集計間隔: 1日
- 表示目盛り: 1日ごと（日曜〜土曜）
- データ本数: 7本
- スタイル: 選択時は青背景・白文字

**6.4: 月ボタン**
- 概要: 月単位でデータを表示
- 文言: 「月」
- データソース: ゴールド層（daily_summary）
- 集計間隔: 1日
- 表示目盛り: 2日飛ばし（奇数日のみ表示）
- データ本数: 月の日数分
- スタイル: 選択時は青背景・白文字

**動作:**
- いずれか1つのみ選択可能（排他選択）
- 選択変更時に集計時間幅の選択肢を更新し、グラフを再描画

---

### (7) 集計時間幅（時単位のみ表示）

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (7) 集計時間幅： [(7.1) 10 ▼] 分                                                        │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**表示条件:** 表示時間単位が「時」の場合のみ表示。日/週/月の場合は非表示。

**7: 集計時間幅ラベル**
- 概要: 集計時間幅の選択
- ラベル: 「集計時間幅：」
- 単位ラベル: 「分」

**7.1: 集計時間幅セレクトボックス**
- 概要: グラフの1バーあたりの時間幅を選択
- 入力タイプ: Select

| 選択肢 | 初期値 |
|--------|--------|
| 1分, 2分, 3分, 5分, 10分, 15分 | 10分 |

**動作:**
- 選択変更時にグラフを再描画
- 表示時間単位が「日/週/月」に変更された場合は非表示

---

### (8) 時間帯 / (9) 更新ボタン

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (8) 時間帯： [(8.1) 📅 2026/02/05 15:32:42] [(9) 🔄]                                    │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**8: 時間帯ラベル**
- 概要: グラフ表示の基準日時を選択
- ラベル: 「時間帯：」

**8.1: 時間帯日時ピッカー**
- 概要: 基準日時を選択
- 入力タイプ: DateTimePicker
- フォーマット: YYYY/MM/DD HH:mm:ss
- アイコン: 📅（カレンダーアイコン）
- 初期値: 現在日時
- 最小値: データ存在開始日
- 最大値: 現在日時

**9: 更新ボタン**
- 概要: 時間帯を現在日時に更新し、グラフを再描画
- アイコン: 🔄（リフレッシュアイコン）
- 入力タイプ: IconButton

**動作:**
- 時間帯変更時にグラフを再描画
- 更新ボタン押下時に時間帯を現在日時に設定し、グラフを再描画
