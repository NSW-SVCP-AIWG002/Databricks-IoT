### (2) マスタ種別選択エリア

**要素構成図:**

```
┌───────────────────────────────────────────────────────────────┐
│ (2) マスタ種別選択エリア                                       │
├───────────────────────────────────────────────────────────────┤
│ (2.1) 対象マスタ                                              │
│ ┌───────────────────────────────────────────────────────────┐ │
│ │ 選択してください                                         ▼ │ │
│ └───────────────────────────────────────────────────────────┘ │
└───────────────────────────────────────────────────────────────┘
```

**2-1: マスタ種別ドロップダウン**
- 概要: インポート対象のマスタ種別を選択
- ラベル: 「対象マスタ」
- プレースホルダー: 「選択してください」
- 選択肢: DB: master_listからユーザーの権限に応じて取得
  - システム保守者: デバイス、ユーザー、組織、アラート設定、デバイス在庫情報
  - 管理者: デバイス、ユーザー、組織、アラート設定、（デバイス在庫情報を除く）
  - 販社ユーザー: デバイス、ユーザー、組織、アラート設定、（デバイス在庫情報を除く）
- 選択肢のソート順
  - DB: master_list.master_idの昇順
- 初期値: 未選択

**選択肢の表示名:**
DB: master_list.master_nameを参照

| マスタ種別 | 表示名 |
|-----------|--------|
| device_master | デバイス |
| user_master | ユーザー |
| organization_master | 組織 |
| alert_setting_master | アラート設定 |
| device_stock_info_master | デバイス在庫情報 |

**バリデーション:**
エラーメッセージは(4) エラーメッセージ表示エリアに表示

| 項目 | バリデーションルール | エラーメッセージ |
|------|---------------------|-----------------|
| 2-1 対象マスタ | 必須 | 「対象マスタを選択してください」 |

### (3) ファイルアップロードエリア

**要素構成図:**

```
┌───────────────────────────────────────────────────────────────────┐
│ (3) ファイルアップロードエリア                                      │
├───────────────────────────────────────────────────────────────────┤
│ ┌───────────────────────────────────────────────────────────────┐ │
│ │                                                               │ │
│ │            (3.1) CSVファイルをドラッグ&ドロップ                 │ │
│ │                         または                                │ │
│ │                (3.2) [ファイルを選択]                          │ │
│ │ ───────────────────────────────────────────────────────────── │ │
│ │          (3.3) 対応形式:CSV(.csv) | 最大サイズ:10MB            │ │
│ │                                                               │ │
│ └───────────────────────────────────────────────────────────────┘ │
└───────────────────────────────────────────────────────────────────┘
```

**3-1: ドラッグ&ドロップフィールド**
- 概要: CSVファイルをドラッグ&ドロップで選択するためのフィールド
- ラベル: 「CSVファイルをドラッグ&ドロップまたは」
- 入力タイプ: file
- 受け入れ形式: `.csv`
- 最大サイズ: 10MB

**3-2: ファイル選択ボタン**
- 概要: ファイル選択ダイアログを開くボタン
- ラベル: 「ファイルを選択」
- スタイル: プライマリボタン

**3-3: ファイル情報テキスト**
- 概要: ファイル形式と最大サイズの情報を表示するヘルプテキスト
- 内容:
  - 対応形式: CSV(.csv)
  - 最大サイズ: 10MB

**バリデーション:**
エラーメッセージは(4) エラーメッセージ表示エリアに表示

| 項目 | バリデーションルール | エラーメッセージ |
|------|---------------------|-----------------|
| 3-1 ドラッグ&ドロップフィールド、3-2 ファイル選択ボタン | 必須 | 「CSVファイルを選択してください」 |
| 3-1 ドラッグ&ドロップフィールド、3-2 ファイル選択ボタン | ファイル拡張子（.csv） | 「CSVファイルを選択してください」 |
| 3-1 ドラッグ&ドロップフィールド、3-2 ファイル選択ボタン | 最大サイズ 10MB | 「ファイルサイズは10MB以下にしてください」 |
| 3-1 ドラッグ&ドロップフィールド、3-2 ファイル選択ボタン | ファイル形式（文字コード、区切り文字、ヘッダー行） | 「CSVファイルの形式が正しくありません」 |
| 3-1 ドラッグ&ドロップフィールド、3-2 ファイル選択ボタン | データ行（0行） | 「CSVファイルにデータがありません」 |

### (4) エラーメッセージ表示エリア

**概要:**
ファイル形式や最大サイズのエラーメッセージを表示するエリア。

**表示メッセージ:**

| メッセージID | 表示内容 | 表示タイミング | 表示スタイル |
|-------------|---------|---------------|-------------|
| ERR_FILE_EXT | CSVファイルを選択してください | ファイル選択時 | エラー（赤色） |
| ERR_FILE_SIZE | ファイルサイズは10MB以下にしてください | ファイル選択時 | エラー（赤色） |
| ERR_FILE_FORMAT | CSVファイルの形式が正しくありません | ファイル選択時 | エラー（赤色） |
| ERR_FILE_EMPTY | CSVファイルにデータがありません | ファイル選択時 | エラー（赤色） |
| ERR_MASTER_TYPE_EXT | 対象マスタを選択してください | インポートボタン押下時 | エラー（赤色） |
| ERR_FILE_EXT | CSVファイルを選択してください | インポートボタン押下時 | エラー（赤色） |
