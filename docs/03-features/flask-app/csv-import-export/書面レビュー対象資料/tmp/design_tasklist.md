  - NFR-PERF-001: Web画面応答性能
- `02-requirements/technical-requirements.md`
  - TR-BE-004: CSVインポート/エクスポート機能の技術仕様
  - TR-SEC-004: ロールベースアクセス制御実装
  - TR-SEC-005: データスコープ制限実装

**把握した主要要件:**
- CSVエクスポート:各マスタ一覧画面からエクスポート、絞り込み条件適用
- CSVインポート: 専用画面（TRF-001）、マスタ種別選択、一括登録・更新
- 権限制御: システム保守者・管理者・販社ユーザのみアクセス可能
- データスコープ制限: ロールに応じたデータアクセス範囲制限
- エラーハンドリング: 形式チェック、重複チェック、必須項目チェック、ロールバック
- ファイル制約: 10MB以下、UTF-8 BOM付き、カンマ区切り
- 対応文字コード: UTF-8（BOM付き/なし）、Shift-JIS、EUC-JP（自動検出）
