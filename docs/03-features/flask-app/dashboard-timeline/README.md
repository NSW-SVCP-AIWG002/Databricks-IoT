## 概要

IoTデバイスのセンサーデータを時系列（折れ線グラフ）で可視化する機能。シルバー層のデータを直接参照し、指定期間内の全データ点をプロットする。2つのセンサー項目を選択して左右のY軸（Dual Y-Axis）に割り当て、異なる単位のデータを同一グラフ上で比較表示できる。Apache EChartsを使用して描画する。

---

## 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| DTL-001 | 時系列グラフ表示 | 指定期間のセンサーデータを折れ線グラフで表示（Dual Y-Axis対応） |
| DTL-002 | 期間選択 | 開始・終了日時を指定してデータ範囲を変更 |
| DTL-003 | CSVエクスポート | 表示中のデータをCSVファイルとしてダウンロード |
| DTL-004 | ガジェット登録 | タイムライングラフガジェットをダッシュボードに登録 |

## アクセス権限

| 機能 | システム保守者 | 管理者 | 販社ユーザ | サービス利用者 |
|------|---------------|--------|-----------|-----------------|
| データ表示 | ○ | ○ | ○ | ○ |
| CSV出力 | ○ | ○ | ○ | ○ |
| ガジェット登録 | ○ | ○ | ○ | ○ |

**凡例**:
- **○**: 利用可能
- **-**: 利用不可

**データスコープ制限:**
- 組織階層（organization_closure）によるデータスコープ制限
- 全ユーザ：自身の組織および配下組織に紐づくデバイスのデータのみ表示可能
