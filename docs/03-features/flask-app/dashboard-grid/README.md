# ダッシュボード表ガジェット機能

## 概要

IoTデバイスから収集したセンサーデータを表形式で可視化する機能。シルバー層（生データ）から指定した日時範囲のデータを取得し、22種類のセンサー項目を一覧表示する。

## 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| DTB-001 | データ表示 | 選択した日時範囲のセンサー生データを表形式で表示 |
| DTB-002 | 日時範囲指定 | 開始日時・終了日時による表示データの絞り込み |
| DTB-003 | CSV出力 | 表示データをCSVファイルとしてダウンロード |
| DTB-004 | ガジェット登録 | 表ガジェットの新規登録 |

## 画面一覧

| 画面ID | 画面名 | URL | 説明 |
|--------|--------|-----|------|
| SCR-DTB-001 | 表ガジェット画面 | /customer-dashboard/table | メイン画面 |
| SCR-DTB-002 | 表ガジェット登録画面 | /customer-dashboard/table/create | 登録画面 |

## データソース

### シルバー層（生データ）
- テーブル：`silver_sensor_data`
- 用途：全表示
- データ粒度：受信時刻（10秒間隔）
- 特徴：集約なしの生データを表示

## 円グラフ・棒グラフとの違い

| 項目 | 円グラフ/棒グラフ | 表 |
|------|------------------|-----|
| センサー選択 | あり（1〜5個） | なし（22項目全て表示） |
| 集約方法 | あり（8種） | なし（生データ） |
| 日時指定 | 基準日時 + 表示単位 | 開始・終了日時（範囲） |
| データソース | Silver/Gold層 | Silver層のみ |
| 表示形式 | グラフ | テーブル |
| デバイスモード | 指定デバイス/ツリー連動 | ツリー選択に依存 |

## アクセス権限

| 機能 | システム保守者 | 管理者 | 販社ユーザ | サービス利用者 |
|------|---------------|--------|-----------|-----------------|
| データ表示 | ○ | ○ | ○ | ○ |
| CSV出力 | ○ | ○ | ○ | ○ |
| ガジェット登録 | ○ | ○ | ○ | ○ |

**凡例**:
- **○**: 利用可能
- **-**: 利用不可

**データスコープ制限:**
- 組織階層（organization_closure）によるデータスコープ制限
- 全ユーザ：自身の組織および配下組織に紐づくデバイスのデータのみ表示可能
