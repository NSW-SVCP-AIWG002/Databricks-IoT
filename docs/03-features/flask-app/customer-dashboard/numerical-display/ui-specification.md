### (1) ガジェットヘッダー

- テキスト配置: 中央揃え
- 背景色:
  - デバイス固定: `#F2C7D5`（ピンク）
  - デバイス可変: `#F2D9C7`（ベージュ）

**1-2: 設定ボタン（⚙）**
- 概要: ガジェットの設定メニューを表示
- 配置: 画面右上
- メニュー項目:
  - ガジェットタイトル変更: ガジェット名を編集
  - ガジェット削除: ガジェットを削除
- 処理詳細: [共通UI仕様書](../common/ui-specification.md)を参照

**要素構成図:**


**2: デバイス名**
- 概要: ガジェット設定の `data_source_config.device_mode` に応じてデバイス名を表示

- テキスト配置: 左揃え

## 数値フォーマット仕様

### 表示フォーマット定義

| 項目 | 仕様 |
|------|------|
| 最大有効桁数 | 10桁（符号・小数点を除いた数字部分） |
| 整数部最大桁数 | 8桁（符号含む最大9文字: `-12345678`） |
| 小数部桁数 | 2桁（固定） |
| 表示最大文字数 | 12文字（`-12345678.90` = 符号1+整数8+小数点1+小数2） |
| 符号 | 負値の場合のみ先頭に `-` を付与。正値に `+` は付与しない |
| 小数点 | 常に表示 |
| ゼロ埋め | 行わない（`0001.23` ではなく `1.23` と表示） |

### 表示例

| 生データ値 | 表示文字列 |
|-----------|-----------|
| `12345.6789` | `12345.68`（四捨五入） |
| `-99.5` | `-99.5` |
| `0.0` | `0` |
| `9999999.99` | `9999999.99` |
| `NULL` | `--` |
| `-0.001` | `0.00`（四捨五入） |

### NULL値・エラー時の表示

| 状況 | 表示内容 |
|------|---------|
| データ未取得（NULL値） | `--` |
| デバイス未選択 | `--` |
| DBクエリエラー | `--` |

#### (1) タイトル

**1: タイトル入力**
- 概要: ガジェットの表示名（`dashboard_gadget_master.gadget_name`）を入力
- ラベル: 「タイトル」
- 入力タイプ: テキストボックス
- 最大文字数: 20文字
- 初期値: 「数値表示」
- 必須: ○
- バリデーション: 空文字不可

---

#### (2) 表示デバイス選択

**2: 表示デバイス選択トグルボタン**
- 概要: 数値表示に使うデバイスの決定方式を選択
- ラベル: 「表示デバイス選択」
- 選択肢:
  - `fixed`: デバイス固定 — 登録時に指定したデバイスのデータを常に表示
  - `variable`: デバイス可変（ダッシュボード設定に従う） — ダッシュボード上部のデバイス選択セレクトボックスに連動
- デフォルト: `variable`（デバイス可変）
- `variable` 選択時: (5) 表示デバイス指定を非表示にする
- 必須: ○

---

#### (3) 表示項目選択1 / (4) 表示項目選択2

**3: 表示項目選択リスト1**
- 概要: 数値表示する測定項目を1つ選択する
- ラベル: 「表示項目選択1」
- 入力タイプ: ラジオボタンリスト（1つのみ選択）
- 表示形式: 選択 | 表示名称 | 単位
- 選択制限: 単一選択
- データソース: 
  - DB: measurement_item_masterを全件取得
  - 表示順: measurement_item_master.measurement_item_idの昇順
- 必須: ○

**4: 表示項目選択リスト2**
- 概要: 数値表示する測定項目を1つ選択する
- ラベル: 「表示項目選択2」
- 入力タイプ: ラジオボタンリスト（1つのみ選択）
- 表示形式: 選択 | 表示名称 | 単位
- 選択制限: 単一選択
- データソース: 
  - DB: measurement_item_masterを全件取得
  - 表示順: measurement_item_master.measurement_item_idの昇順
- 必須: ×（未選択の場合、数値表示ガジェットで測定項目名2と数値表示エリア2は表示されない）

---

#### (5) 表示デバイス指定（デバイス固定モード時のみ）

**5.1: 組織フィルター**
- 概要: デバイス選択の絞り込み用組織選択
- 入力タイプ: セレクトボックス
- データソース: ユーザーがアクセス可能な組織一覧
- 動作: 選択変更時にデバイス選択の選択肢を更新

**5.2: デバイス選択**
- 概要: 表示対象のデバイスを選択
- 入力タイプ: セレクトボックス
- データソース: 組織フィルターで絞り込まれたデバイス一覧

**6: デバイス名**
- 概要: 選択されたデバイス名を表示
- 未選択時: 「-」を表示

---

#### (7) ガジェットイメージ

**7: ガジェットイメージ**
- 概要: 数値表示ガジェット種別のプレビュー画像を表示する読み取り専用の領域
- 画像パス: `gadget_type_master.gadget_image_path`（`numerical-display` 種別のレコード）
- 入力操作: なし（読み取り専用）

---

#### (8) グループ選択

**8: グループ選択**
- 概要: ガジェットを配置するダッシュボードグループを選択
- 入力タイプ: セレクトボックス
- データソース: dashboard_group_master（データスコープ制限あり）
- 選択肢のソート順: dashboard_group_master.display_order の昇順
- 初期値: 選択肢の先頭値

---

#### (9) 登録ボタン / (10) キャンセルボタン

**9: 登録ボタン**
- 概要: フォーム送信で数値表示ガジェットを登録する
- ラベル: 「登録」
- スタイル: プライマリボタン（青系）
- クリック時: バリデーション実行 → 登録 API 呼び出し

**10: キャンセルボタン**
- 概要: 入力を破棄してモーダルを閉じる
- ラベル: 「キャンセル」
- スタイル: セカンダリボタン

---
