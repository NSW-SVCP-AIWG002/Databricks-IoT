### 要素詳細

### (1) ヘッダー

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (1)                                                                                [⚙]  │
│                                      ガジェット名                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**1: ガジェット名**
- 概要: ガジェット名を表示
- スタイル: 中央揃え

**1.1: 設定アイコン（⚙）**
- 概要: ガジェットの設定メニューを表示
- 配置: 画面右上
- メニュー項目:
  - ガジェットタイトル変更: ガジェット名を編集
  - ガジェット削除: ガジェットを削除
- 処理詳細: [共通UI仕様書](../common/ui-specification.md)を参照

### (2) デバイス名 / (3) CSVエクスポートボタン

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (2) デバイス名                                                    (3) [CSVエクスポート]  │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**2: デバイス名**
- 概要: 現在表示中のデバイス名を表示
- 配置: 左寄せ

**3: CSVエクスポートボタン**
- 概要: 現在表示中のグラフデータをCSVファイルとしてダウンロード
- 文言: 「CSVエクスポート」
- スタイル:プライマリボタン（青色）
- 配置: 右寄せ
- 出力ファイル名: `sensor_data_{yyyyMMddHHmmss}.csv`

---

### (4) 開始日時ピッカー

- ラベル: 「開始日時：」
- 入力タイプ: 日時ピッカー（カレンダー + 時刻）
- 形式: YYYY/MM/DD HH:mm:ss
- 初期値: 現在日時 - 1時間
- 最大値: 終了日時
- 配置: (5) 終了日時ピッカーと横並び（左側）

**(4.1) 開始日時更新ボタン**
- アイコン: 🔄
- 入力タイプ: IconButton
- 動作: 押下時に現在日時 - 1時間を開始日時に設定してグラフを再描画

---

### (5) 終了日時ピッカー

- ラベル: 「終了日時：」
- 入力タイプ: 日時ピッカー（カレンダー + 時刻）
- 形式: YYYY/MM/DD HH:mm:ss
- 初期値: 現在日時
- 最大値: 現在日時
- 配置: (4) 開始日時ピッカーと横並び（右側）

**日時範囲の制約:**
- 開始日時 < 終了日時 であること
- 終了日時 - 開始日時 ≤ 24時間

**(5.1) 終了日時更新ボタン**
- アイコン: 🔄
- 入力タイプ: IconButton
- 動作: 押下時に現在日時を終了日時に設定してグラフを再描画

---

| 2x2 | 2x2 | 標準サイズ |
| 2x4 | 2x4  | 横長サイズ |

---

#### (7) 登録ボタン

- ラベル: 「登録」
- スタイル: プライマリボタン
- 配置: 下部中央
- 動作: ガジェット登録APIを呼び出し

**登録成功時:**
- モーダルを閉じる
- ダッシュボードに新規ガジェットを追加
- 成功モーダル表示

**登録失敗時:**
- エラーメッセージを表示
- モーダルは閉じない

---

#### (8) キャンセルボタン

- ラベル: 「キャンセル」
- スタイル: セカンダリボタン
- 配置: モーダル右下（登録ボタン隣）
- 動作: モーダルを閉じてダッシュボードに戻る（入力内容は破棄）
