## 概要

IoTデバイスから収集したセンサーデータを円グラフで可視化する機能です。顧客作成ダッシュボード画面に埋め込んで表示します。シルバー層から最新のセンサーデータを取得し、複数項目の値の割合を視覚的に確認可能です。

## データソース

### シルバー層（最新データ取得）

- テーブル: `silver_sensor_data`
- 用途: 最新センサー値の表示
- データ粒度: 受信時刻
- 取得方法: デバイスごとの最新レコード

## Flaskルート一覧

| No | ルート名 | エンドポイント | メソッド | 用途 | レスポンス形式 |
|----|---------|---------------|---------|------|---------------|

| 2 | ガジェットデータ取得 | `/analysis/customer-dashboard/gadgets/<gadget_uuid>/data` | POST | ガジェットのグラフ表示用データ取得 | JSON (AJAX) |
| 3 | ガジェット登録画面 | `/analysis/customer-dashboard/gadgets/circle-chart/create` | GET | 円グラフガジェット登録モーダル表示 | HTML（モーダル） |
| 4 | ガジェット登録実行 | `/analysis/customer-dashboard/gadgets/circle-chart/register` | POST | 円グラフガジェット登録処理 | JSON (AJAX) |

## アクセス権限

| 機能 | システム保守者 | 管理者 | 販社ユーザ | サービス利用者 |
|------|---------------|--------|-----------|---------------|
| ガジェット表示 | ○ | ○ | ○ | ○ |
| ガジェット登録 | ○ | ○ | ○ | ○ |

**凡例:**
- **○**: 利用可能
- **-**: 利用不可

**データスコープ制限**:
- **すべてのユーザー**: 組織階層（organization_closure）によるデータスコープ制限
  - ユーザーの所属組織とその下位組織に属するデータのみアクセス可能
