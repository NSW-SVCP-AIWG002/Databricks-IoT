## 要素詳細

### (1) ヘッダー

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (1)                                                                                [⚙] │
│                                      ガジェット名                                        │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**1-1: ガジェット名**
- 概要: `dashboard_gadget_master.gadget_name`の値を表示
- テキスト配置: 中央揃え
- 背景色:
  - デバイス固定: `#F2C7D5`（ピンク）
  - デバイス可変: `#F2D9C7`（ベージュ）

**1-2: 設定ボタン（⚙）**
- 概要: ガジェットの設定メニューを表示
- 配置: 画面右上
- メニュー項目:
  - ガジェットタイトル変更: ガジェット名を編集
  - ガジェット削除: ガジェットを削除
- 処理詳細: [共通UI仕様書](../common/ui-specification.md)を参照

---

### (2) デバイス名 / (3) CSVエクスポートボタン

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (2) デバイス名                                                      (3) [CSVエクスポート] │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**2: デバイス名**
- 概要: ガジェット設定の `data_source_config.device_id` に応じてデバイス名を表示
  - `device_id != null`: `data_source_config.device_id` のデバイス名を表示
  - `device_id == null`: `dashboard_user_setting.device_id` のデバイス名を表示
- デバイスが未選択（`device_id == NULL`）の場合: `--` を表示
- テキスト配置: 左揃え

**3: CSVエクスポートボタン**
- 概要: 現在表示中のグラフデータをCSVファイルとしてダウンロード
- ラベル: 「CSVエクスポート」
- スタイル: プライマリボタン（青色）
- 配置: 右揃え
- 出力ファイル名: `sensor_data_{yyyyMMddHHmmss}.csv`

---

```
└─────────────────────────────────────────────────────────────────────────────────────┘
```

**4: 測定項目名**
- 概要: `measurement_item_master.display_name` と `measurement_item_master.unit_name` の値を表示
- テキスト配置: 中央揃え
- 形式: 表示名（単位）

**5: 棒グラフ（年）**
- 概要: 選択した年から直近3年のゴールド層（月次サマリ）データを年度別に棒グラフで表示
- 描画方式: Apache ECharts
- X軸: 時間軸（基準月から12ヶ月分）
- Y軸: 値（左側に表示、選択項目の単位）
- バーの色:
  - 選択年から2年前: #E8877F
  - 選択年から1年前: #7CB5D2
  - 選択年: #A8D88E

**6: 凡例**
- 概要: グラフ表示期間を表示
- 表示形式: ■ YYYY/MM~YYYY/MM
- 配置: グラフ下部中央
- 例: 「■ 2024/01~2024/12」

### (7) 年選択 / (8) 基準月選択

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                        (7) 年： [2026 ▼]  (8) 基準月： [1 ▼]                             │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**7: 年選択**
- 概要: グラフ表示期間の基準年を選択
- ラベル: 「年：」
- 選択肢: 現在年を含む直近10年
- デフォルト: 現在年

**8: 基準月選択** 
- 概要: グラフ表示期間の基準月を選択
- ラベル: 「基準月：」
- 選択肢: 12ヶ月
- デフォルト: 1

**グラフ表示期間の算出方法:**
- 開始年: 選択年-2年
- 開始月: 基準月
- 終了年: 選択年+1年
- 終了月: 基準月-1月（1月は12月）
- 例: 選択年が `2026`、基準月が `6月` の場合、グラフ表示期間が `2024年6月~2027年5月`

| (12) | 登録ボタン | register_button | Button | I | - | - | - | 固定値: 「登録」 | プライマリボタン  |
| (13) | キャンセルボタン | cancel_button | Button | I | - | - | - | 固定値: 「キャンセル」 | セカンダリボタン  |

**凡例**:
- **必須** ○: 常に必須 / △: 条件付き必須（デバイス固定モード選択時のみ必須）

### 要素詳細（登録画面）

#### (1) タイトル

**1: タイトル入力**
- 概要: ガジェットの表示名（`dashboard_gadget_master.gadget_name`）を入力
- ラベル: 「タイトル」
- 入力タイプ: テキストボックス
- 最大文字数: 20文字
- 初期値: 「棒グラフ（年）」
- 必須: ○
- バリデーション: 空文字不可

---

#### (2) 表示デバイス選択

**2: 表示デバイス選択トグルボタン**
- 概要: 数値表示に使うデバイスの決定方式を選択
- ラベル: 「表示デバイス選択」
- 選択肢:
  - `fixed`: デバイス固定 — 登録時に指定したデバイスのデータを常に表示
  - `variable`: デバイス可変（ダッシュボード設定に従う） — ダッシュボード上部のデバイス選択セレクトボックスに連動
- デフォルト: `variable`（デバイス可変）
- `variable` 選択時: (5) 表示デバイス指定を非表示にする
- 必須: ○

---

#### (3) 表示項目選択 / (3.1) 表示項目選択リスト

**3: 表示項目選択リスト**
- 概要: 棒グラフ（年）に表示するセンサー項目を1つ選択
- ラベル: 「表示項目選択」
- 入力タイプ: ラジオボタンリスト（1つのみ選択）
- 表示形式: 選択 | 表示名称 | 単位
- 選択制限: 単一選択
- データソース: 
  - DB: measurement_item_masterを全件取得
  - 表示順: measurement_item_master.measurement_item_idの昇順
- 必須: ○

---

#### (4) 表示デバイス指定（デバイス固定モード時のみ）

**4.1: 組織フィルター**
- 概要: デバイス選択の絞り込み用組織選択
- 入力タイプ: セレクトボックス
- データソース: ユーザーがアクセス可能な組織一覧
- 動作: 選択変更時にデバイス選択の選択肢を更新

**4.2: デバイス選択**
- 概要: 表示対象のデバイスを選択
- 入力タイプ: セレクトボックス
- データソース: 組織フィルターで絞り込まれたデバイス一覧

**5: デバイス名**
- 概要: 選択されたデバイス名を表示
- 未選択時: 「-」を表示

---

#### (6) ガジェットイメージ

**6: ガジェットイメージ**
- 概要: 棒グラフ（年）ガジェット種別のプレビュー画像を表示する読み取り専用の領域
- 画像パス: `gadget_type_master.gadget_image_path`（`bar-chart-year` 種別のレコード）
- 入力操作: なし（読み取り専用）

---

#### (7) グループ選択 / (8) 集約方法

**7: グループ選択**
- 概要: ガジェットを配置するダッシュボードグループを選択
- 入力タイプ: セレクトボックス
- データソース: dashboard_group_master（データスコープ制限あり）
- 選択肢のソート順: dashboard_group_master.display_order の昇順
- 初期値: 選択肢の先頭値

**8: 集約方法**
- 概要: 複数データの集約方法を選択
- 入力タイプ: セレクトボックス
- データソース: gold_summary_method_master（delete_flag = FALSE）
- 選択肢のソート順: gold_summary_method_master.summary_method_id の昇順
- 初期値: 選択肢の先頭値
- 選択肢:

| summary_method_id | summary_method_code | 集約方法名 |
|-------------------|---------------------|------------|
| 1 | AVG | 平均値 |
| 2 | MAX | 最大値 |
| 3 | MIN | 最小値 |
| 4 | P25 | 第1四分位数 |
| 5 | MEDIAN | 中央値 |
| 6 | P75 | 第3四分位数 |
| 7 | STDDEV | 標準偏差 |
| 8 | P95 | 上側5％境界値 |
| 9 | SUM | 合計 |

---

#### (9) 最小値 / (10) 最大値

**9: 最小値**
- 概要: Y軸の最小値を入力
- 入力タイプ: テキストボックス（数値）
- 最大桁数: 10桁（小数点含む）
- 必須: -（未入力時は自動スケール）
- バリデーション: 数値のみ、最大値より小さい値

**10: 最大値**
- 概要: Y軸の最大値を入力
- 入力タイプ: テキストボックス（数値）
- 最大桁数: 10桁（小数点含む）
- 必須: -（未入力時は自動スケール）
- バリデーション: 数値のみ、最小値より大きい値

---

#### (11) 部品サイズ

**11: 部品サイズ**
- 概要: ガジェットの表示サイズを選択
- 入力タイプ: セレクトボックス
- 必須: ○
- 初期値: 0（2×2）
- 選択肢:

| 値 | ラベル | 説明 |
|----|--------|------|
| 0 | 2x2 | 標準サイズ |
| 1 | 2x4 | 横長サイズ |

---

#### (12) 登録ボタン / (13) キャンセルボタン

**12: 登録ボタン**
- 概要: フォーム送信で数値表示ガジェットを登録する
- ラベル: 「登録」
- スタイル: プライマリボタン（青系）
- クリック時: バリデーション実行 → 登録 API 呼び出し

**13: キャンセルボタン**
- 概要: 入力を破棄してモーダルを閉じる
- ラベル: 「キャンセル」
- スタイル: セカンダリボタン
