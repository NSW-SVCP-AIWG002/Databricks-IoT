### (2) デバイス名 / (3) CSVエクスポートボタン

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ (2) デバイス名                                                    (3) [CSVエクスポート]  │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

**2: デバイス名**
- 概要: 現在表示中のデバイス名を表示
- 配置: 左寄せ

**3: CSVエクスポートボタン**
- 概要: 現在表示中のグラフデータをCSVファイルとしてダウンロード
- 文言: 「CSVエクスポート」
- スタイル:プライマリボタン（青色）
- 配置: 右寄せ

**CSVファイル仕様:**

| 項目 | 内容 |
|------|------|
| ファイル名形式 | `sensor_data_{yyyyMMddHHmmss}.csv` |
| 文字コード | UTF-8（BOM付き） |

**CSV列構成:**

| 列番号 | ヘッダー名 | カラム名 | 説明 |
|--------|-----------|---------|------|
| 1 | event_timestamp | 受信日時 | 受信日時（YYYY-MM-DD HH:mm:ss形式） |
| 2 | device_name | デバイス名 | デバイス名称 |
| 3 | {左表示項目ラベル} | {左表示項目カラム名} | 左表示項目として選択された値 |
| 4 | {右表示項目ラベル} | {右表示項目カラム名} | 右表示項目として選択された値 |

**注:** CSV列は選択された2つの表示項目のみを出力する（全22項目ではない）。

### (4) 折れ線グラフエリア

**要素構成図**

```
┌─────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ (4) 折れ線グラフエリア                                                                                │
│                                                                                                      │
│  左Y軸 ┃                                                             ┃ 右Y軸                          │
│  [単位] ┃         ╱╲    ╱╲                                           ┃ [単位]                         │
│         ┃        ╱  ╲  ╱  ╲        ╱╲                               ┃                                │
│         ┃  ╱╲  ╱    ╲╱    ╲  ╱╲  ╱  ╲                             ┃                                  │
│         ┃ ╱  ╲╱             ╲╱  ╲╱    ╲                             ┃                                │
│         ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛                                    │
│                                    X軸（時間）                                                        │
└──────────────────────────────────────────────────────────────────────────────────────────────────────┘
```

**4: 折れ線グラフ**
- 概要: センサーデータを折れ線グラフで時系列表示
- 描画方式: Apache ECharts
- X軸: 時間軸（`sensor_data_view.event_timestamp`）

**Dual Y-Axis仕様:**

| Y軸 | 位置 | 説明 |
|-----|------|------|
| 左Y軸 | グラフ左側 | 1番目に選択された表示項目の値を表示 |
| 右Y軸 | グラフ右側 | 2番目に選択された表示項目の値を表示 |

**Y軸設定:**

| 設定項目 | 値 | 備考 |
|---------|-----|------|
| splitNumber | 10 | 目盛り10分割 |
| min | ユーザー指定値（任意） | 未設定時はECharts自動計算 |
| max | ユーザー指定値（任意） | 未設定時はECharts自動計算 |
| axisLabel | 表示項目の単位を表示 | ℃、rpm、% |

**最小値/最大値設定の動作:**
- min/maxが設定されている場合、EChartsの `yAxis.min` / `yAxis.max` に値を設定
- 範囲外のデータ点はグラフ上に描画されない（EChartsの標準動作）
- 左Y軸・右Y軸それぞれ独立して設定可能

**ツールチップ表示形式:**
```
2026/02/17 08:30:00
{左項目名}: 25.5 ℃
{右項目名}: 2500 rpm
```

**データなし時の表示:**
- 空のグラフを表示

#### (11) 開始日時ピッカー

- ラベル: 「開始日時：」
- 入力タイプ: 日時ピッカー（カレンダー + 時刻）
- 形式: YYYY/MM/DD HH:mm:ss
- 初期値: 現在日時 - 1時間
- 最大値: 終了日時
- 配置: (12) 終了日時ピッカーと横並び（左側）

**(11.1) 開始日時更新ボタン**
- アイコン: 🔄
- 入力タイプ: IconButton
- 動作: 押下時に現在日時 - 1時間を開始日時に設定してグラフを再描画

---

#### (12) 終了日時ピッカー

- ラベル: 「終了日時：」
- 入力タイプ: 日時ピッカー（カレンダー + 時刻）
- 形式: YYYY/MM/DD HH:mm:ss
- 初期値: 現在日時
- 最大値: 現在日時
- 配置: (11) 開始日時ピッカーと横並び（右側）

**日時範囲の制約:**
- 開始日時 < 終了日時 であること
- 終了日時 - 開始日時 ≤ 24時間

**(12.1) 終了日時更新ボタン**
- アイコン: 🔄
- 入力タイプ: IconButton
- 動作: 押下時に現在日時を終了日時に設定してグラフを再描画

### 要素詳細（登録画面）

#### (1) タイトル

**1: タイトル入力**
- 概要: ガジェットのタイトルを入力
- 入力タイプ: テキストボックス
- 最大文字数: 20文字
- 初期値: 「時系列」
- 必須: ○
- バリデーション: 空文字不可

---

#### (2) 表示デバイス選択

**2.1: デバイス固定ボタン**
- 概要: 特定のデバイスを指定してデータを表示
- スタイル: 選択時は青背景・白文字
- 初期値: 選択状態
- 動作: 選択時に表示デバイス指定エリア(5)を表示

**2.2: デバイス可変ボタン**
- 概要: ダッシュボードのデータソース選択フォームに連動してデータを表示
- 特徴: データソース選択フォーム上でデバイスを選択することでデータを動的に切り替え可能
- 注意: データソース選択フォーム上でデバイスが選択されていない場合はデータを取得できない
- スタイル: 選択時は青背景・白文字
- 動作: 選択時に表示デバイス指定エリア(5)を非表示

---

#### (3) 左表示項目選択 / (3.1) 左表示項目選択リスト / (4) 右表示項目選択 / (4.1) 右表示項目選択リスト

**3, 4: 表示項目選択ラベル**
- 概要: センサー選択リストのセクションラベルを表示
- 文言: 「表示項目選択」
- 表示条件: 両モード（デバイス固定/デバイス可変）共通で表示

**3.1, 4.1: 表示項目選択リスト**
- 概要: 時系列グラフに表示するセンサー項目を選択
- 入力タイプ: ラジオボタンリスト（1つのみ選択）
- 表示形式: 選択 | 表示名称 | 単位
- 選択制限: 単一選択
- データソース: 凡例一覧（22項目）

---

#### (5) 表示デバイス指定（デバイス固定モード時のみ）

**5.1: 組織フィルター**
- 概要: デバイス選択の絞り込み用組織選択
- 入力タイプ: セレクトボックス
- データソース: ユーザーがアクセス可能な組織一覧
- 動作: 選択変更時にデバイス選択の選択肢を更新

**5.2: デバイス選択**
- 概要: 表示対象のデバイスを選択
- 入力タイプ: セレクトボックス
- データソース: 組織フィルターで絞り込まれたデバイス一覧

**6: デバイス名**
- 概要: 選択されたデバイス名を表示
- 未選択時: 「-」を表示

---

#### (7) ガジェットイメージ

**7: 時系列グラフガジェットイメージ**
- 概要: 時系列グラフガジェットのイメージを表示

---

#### (8) 左表示項目 最小値/最大値 / (9) 右表示項目 最小値/最大値

**8, 9: 最小値**
- 概要: Y軸の最小値を入力
- 入力タイプ: テキストボックス（数値）
- 最大桁数: 10桁（小数点含む）
- 必須: -（未入力時は自動スケール）
- バリデーション: 数値のみ、最大値より小さい値

**8, 9: 最大値**
- 概要: Y軸の最大値を入力
- 入力タイプ: テキストボックス（数値）
- 最大桁数: 10桁（小数点含む）
- 必須: -（未入力時は自動スケール）
- バリデーション: 数値のみ、最小値より大きい値

---

#### (10) グループ選択

**10: グループ選択**
- 概要: ガジェットを配置するダッシュボードグループを選択
- 入力タイプ: セレクトボックス
- データソース: dashboard_group_master（データスコープ制限あり）

---

#### (11) 部品サイズ

**11: 部品サイズ**
- 概要: ガジェットの表示サイズを選択
- 入力タイプ: セレクトボックス
- 必須: ○
- 選択肢:

| 値 | ラベル | 説明 |
|----|--------|------|
| 2x2 | 2x2 | 標準サイズ |
| 2x4 | 2x4 | 横長サイズ |

---

#### (12) 登録ボタン / (13) キャンセルボタン

**12: 登録ボタン**
- 概要: ガジェットを登録
- スタイル: 青背景、白文字（プライマリボタン）
- 動作: バリデーション後、ガジェット情報をサーバに送信

**13: キャンセルボタン**
- 概要: ガジェット登録モーダルを閉じる
- 文言: 「キャンセル」
- スタイル: グレー背景（セカンダリボタン）
- 動作: モーダルを閉じてダッシュボードに戻る（入力内容は破棄）
