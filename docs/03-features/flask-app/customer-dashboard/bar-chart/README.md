# ダッシュボード棒グラフ機能

## 概要

IoTデバイスから収集したセンサーデータを棒グラフで可視化する機能。シルバー層（生データ）またはゴールド層（集計データ）のデータを時間単位（時/日/週/月）で表示し、センサー値の推移を確認できる。

## 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| DBG-001 | グラフ表示 | 選択した項目・期間のデータを棒グラフで表示 |
| DBG-002 | 表示単位切替 | 時/日/週/月単位での表示切替 |
| DBG-003 | 期間選択 | 日時ピッカーによる表示期間の指定 |
| DBG-004 | 集計間隔選択 | データの集計間隔を選択 |
| DBG-005 | CSVエクスポート | 表示データをCSVファイルとしてダウンロード |
| DBG-006 | ガジェット登録 | 棒グラフガジェットの新規登録 |

## データソース

### シルバー層（時単位表示）
- テーブル：`silver_sensor_data`
- 用途：時単位表示
- データ粒度：受信時刻（10秒間隔）
- 集計時間幅：1分/2分/3分/5分/10分/15分（選択可能）

### ゴールド層（日単位表示）
- テーブル：**TODO**（新規時間単位サマリーテーブル）
- 用途：日単位表示
- データ粒度：1時間単位で集計済み
- 表示目盛り：2時間ごと（0時, 2時, ..., 22時）

### ゴールド層（週/月単位表示）
- テーブル：`gold_sensor_data_daily_summary`
- 用途：週/月単位表示
- データ粒度：1日単位で集計済み
- 週単位：日曜〜土曜の7日間
- 月単位：月の全日分（表示目盛りは奇数日）

## 対象センサー項目

22種類のセンサー項目を表示可能：
- 温度系：外気温、冷凍/冷蔵設定温度、内部温度、吸込/吹出/霜取温度
- 機器系：コンプレッサーRPM、ファンモーターRPM、ヒーター出力
- その他：通信通電時間、バッテリー電圧

| ガジェット登録 | ○ | ○ | ○ | ○ |

**凡例**:
- **○**: 利用可能
- **-**: 利用不可

**データスコープ制限:**
- 組織階層（organization_closure）によるデータスコープ制限
- 全ユーザ：自身の組織および配下組織に紐づくデバイスのデータのみ表示可能
