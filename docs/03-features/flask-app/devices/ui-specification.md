## 基本情報

| 項目         | 内容                                                           |
| ------------ | -------------------------------------------------------------- |
| 画面ID       | `ADM-001`, `ADM-002`, `ADM-003`, `ADM-004`                     |
| 画面名       | デバイス管理画面                                               |
| 画面パス     | `/admin/devices`                                               |
| アクセス権限 | システム保守者、管理者、販社ユーザ、サービス利用者（参照のみ） |
| 親画面       | グローバルメニュー（管理機能）                                 |
| 子画面       | なし                                                           |

### レイアウト構成

画面の主要なレイアウト領域を説明します。

| No  | 領域                      | 説明                                     | 備考               |
| --- | ------------------------- | ---------------------------------------- | ------------------ |
| (1) | 画面タイトル              | 画面名とナビゲーション                   | 固定表示           |
| (2) | 検索フォーム              | 検索条件入力エリア                       | 折りたたみ可能     |
| (3) | アクションボタン          | 登録・削除・エクスポートボタン           | 画面右上に配置     |

| (5) | 検索結果件数表示          | 現在の検索条件でヒットした件数を表示     |                    |
| (6) | データテーブル            | デバイス一覧表示（ページネーション含む） | スクロール可能     |
| (7) | デバイス登録/更新モーダル | デバイス登録・更新フォーム               | モーダルダイアログ |
| (8) | デバイス参照モーダル      | デバイス詳細情報表示                     | モーダルダイアログ |
| (9) | 削除確認モーダル          | 削除前の確認ダイアログ                   | モーダルダイアログ |

- 画面サイズは1920×1080を基準に設計
- 画面左部にグローバルメニューカラムを表示
- 登録系のボタンは青系、削除ボタンは赤系を使用
- 登録、更新、参照画面は同画面内でモーダルとして表示
- ページネーション形式: 「1 … n-1 n n+1 … 最大ページ」
