## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | `IDS-001` |
| 画面名 | 店舗モニタリング画面 |
| 画面パス | `/industry-dashboard/store-monitoring` |
| アクセス権限 | システム保守者、管理者、販社ユーザ、サービス利用者（すべてのロール） |

| 子画面 | IDS-002（デバイス詳細画面） |

- 店舗モニタリング画面はログイン後の初期画面（ホーム画面）として機能します
