## その他

### 備考

- NSWは自分自身以外の子組織を持たない。また、自分自身以外の子組織にならない。
- 出荷ごとに操作が必要になり、NSWの運用負荷が上がる。
- 販社ユーザがサービス利用会社に振り分ける際の管理は本システムの対象外とする。
- 顧客が用意したデバイスの場合、デバイス在庫IDは空でデバイスマスタに登録（台帳マスタには何も登録されない）。
- 台帳マスタのモデル情報、MACアドレスはオリジナルの値を履歴として残しておくため設置する。

### 他のユースケース候補

- 在庫状況が「出荷予定」か「出荷済み」のときのみ、デバイスマスタにも情報を登録する
