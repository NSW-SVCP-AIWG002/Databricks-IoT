## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | `ADM-013` 〜 `ADM-022` |
| 画面名 | デバイス台帳管理画面 |
| 画面パス | `/admin/device-inventory` |
| アクセス権限 | システム保守者のみ |
| 親画面 | グローバルメニュー（管理機能） |
| 子画面 | ADM-014 (登録), ADM-015 (更新), ADM-016 (参照), ADM-017（登録確認）, ADM-018（更新確認）, ADM-019（削除確認）, ADM-020（登録完了）, ADM-021（更新完了）, ADM-022（削除完了） |

| (9) | デバイス台帳削除確認モーダル | 削除確認 | モーダルダイアログ（読み取り専用） |
| (10) | デバイス台帳登録完了モーダル | 登録完了通知 | モーダルダイアログ（読み取り専用） |
| (11) | デバイス台帳更新完了モーダル | 更新完了通知 | モーダルダイアログ（読み取り専用） |
| (12) | デバイス台帳削除完了モーダル | 削除完了通知 | モーダルダイアログ（読み取り専用） |

**2.1: デバイスUUID入力**
- 概要: クラウドに登録するデバイスID（デバイスUUID）で部分一致検索
- ラベル: 「クラウドに登録するデバイスID」
- プレースホルダー: なし
- 入力中のリアルタイムバリデーション: なし

**2.2: デバイス名入力**
- 概要: デバイス名で部分一致検索
- ラベル: 「デバイス名」
- プレースホルダー: なし
- 入力中のリアルタイムバリデーション: なし

**2.3: デバイス種別ドロップダウン**
- 概要: デバイスの種別でフィルタリング
- ラベル: 「デバイス種別」
- 選択肢: マスタデータから取得（デフォルト: すべて）

**2.4: 在庫状況ドロップダウン**
- 概要: 在庫状況でフィルタリング
- ラベル: 「在庫状況」
- 選択肢: マスタデータから取得（デフォルト: すべて）

**2.5: 在庫場所入力**
- 概要: 在庫場所で部分一致検索
- ラベル: 「在庫場所」
- プレースホルダー: なし
- 入力中のリアルタイムバリデーション: なし

**2.6, 2.7: 購入日範囲**
- 概要: 購入日の範囲でフィルタリング
- ラベル: 「購入日」（開始日）/（終了日）
- 形式: 日付ピッカー（YYYY/MM/DD）

**2.8: ソート項目ドロップダウン**
- 概要: 検索結果のソート基準を選択
- ラベル: 「ソート項目」
- 選択肢:
  - 未選択 (デフォルト(デバイス在庫IDがソート項目になる))
  - クラウド登録するデバイスID
  - デバイス名
  - デバイス種別
  - SIMID
  - MACアドレス
  - 在庫状況
  - 購入日
  - 保証期限
  - 在庫場所

**2.9: ソート順ドロップダウン**
- 概要: ソートの昇順・降順を選択
- ラベル: 「ソート順」
- 選択肢:
  - 未選択 (デフォルト(降順がソート順になる))
  - 昇順
  - 降順

**2.10: 検索ボタン**
- 概要: 入力した条件で検索を実行
- ラベル: 「検索」
- スタイル: プライマリボタン

**バリデーション:**

| 項目 | バリデーションルール | エラーメッセージ |
|------|---------------------|-----------------|
| 2.1 デバイスUUID | 最大128文字 | 「クラウド登録するデバイスIDは128文字以内で入力してください」 |
| 2.2 デバイス名 | 最大100文字 | 「デバイス名は100文字以内で入力してください」 |
| 2.5 在庫場所 | 最大100文字 | 「在庫場所は100文字以内で入力してください」 |
| 2.6, 2.7 購入日 | 開始日 ≤ 終了日 | 「開始日は終了日以前を指定してください」 |

**4.1: クラウドに登録するデバイスID**

- 入力タイプ: text
- 登録先: device_master.device_uuid

**4.2: デバイス名**
- 概要: デバイスの名称を入力
- ラベル: 「デバイス名」
- 入力タイプ: text
- 登録先: device_master.device_name

**4.3: デバイス種別**
- 概要: デバイスの種類を選択
- ラベル: 「デバイス種別」
- 選択肢: device_type_masterテーブルから取得
- 登録先: device_master.device_type_id

**4.4: モデル情報**
- 概要: デバイスのモデル名・型番を入力（台帳マスタにはオリジナル値として保持）
- ラベル: 「モデル情報」
- 入力タイプ: text
- 登録先: device_inventory_master.device_model, device_master.device_model

**4.5: SIMID**
- 概要: デバイスに搭載されているSIMカードのIDを入力
- ラベル: 「SIMID」
- 入力タイプ: text
- 登録先: device_master.sim_id

**4.6: MACアドレス**
- 概要: ネットワークインターフェースの物理アドレス（台帳マスタにはオリジナル値として保持）
- ラベル: 「MACアドレス」
- プレースホルダー: なし
- 入力補助: 2文字ごとに自動でコロン挿入
- 登録先: device_inventory_master.mac_address, device_master.mac_address

**4.7: 組織**
- 概要: デバイスが所属する組織を選択
- ラベル: 「組織」
- 選択肢: organization_masterテーブルから取得（NSW配下組織すべて）
- 登録先: device_master.organization_id

**4.8: ソフトウェアバージョン**
