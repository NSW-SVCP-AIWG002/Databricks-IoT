## 要素詳細

### (2) 検索フォーム

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ (2) 検索フォーム                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│ (2.1) キーワード        (2.2) デバイス種別    (2.3) 在庫状況               │
│ ┌─────────────────────┐ ┌─────────────────┐ ┌─────────────────┐            │
│ │                     │ │▼ すべて         │ │▼ すべて         │            │
│ └─────────────────────┘ └─────────────────┘ └─────────────────┘            │
│                                                                             │
│ (2.4) 在庫場所          (2.5)〜(2.6) 購入日範囲                            │
│ ┌─────────────────────┐ ┌────────────┐  〜  ┌────────────┐                │
│ │▼ すべて             │ │📅          │      │📅          │                │
│ └─────────────────────┘ └────────────┘      └────────────┘                │
│                                                                             │
│                                         (2.7) [検索]  (2.8) [クリア]       │
└─────────────────────────────────────────────────────────────────────────────┘
```

**2-1: キーワード入力**
- 概要: デバイスIDまたはデバイス名で部分一致検索
- ラベル: 「キーワード」
- プレースホルダー: 「デバイスIDまたはデバイス名で検索」
- 入力中のリアルタイムバリデーション: なし

**2-2: デバイス種別ドロップダウン**
- 概要: デバイスの種別でフィルタリング
- ラベル: 「デバイス種別」
- 選択肢:
  - すべて（デフォルト）
  - センサー
  - ゲートウェイ
  - コントローラー
  - その他

**2-3: 在庫状況ドロップダウン**
- 概要: 在庫状況でフィルタリング
- ラベル: 「在庫状況」
- 選択肢:
  - すべて（デフォルト）
  - 在庫中
  - 出荷済み
  - 修理中
  - 廃棄予定
  - 廃棄済み

**2-4: 在庫場所ドロップダウン**
- 概要: 保管場所でフィルタリング
- ラベル: 「在庫場所」
- 選択肢: マスタデータから動的取得

**2-5, 2-6: 購入日範囲**
- 概要: 購入日の範囲でフィルタリング
- ラベル: 「購入日」
- 形式: 日付ピッカー（YYYY/MM/DD）

**バリデーション:**

| 項目 | バリデーションルール | エラーメッセージ |
|------|---------------------|-----------------|
| 2-1 キーワード | 最大100文字 | 「キーワードは100文字以内で入力してください」 |
| 2-5, 2-6 購入日 | 開始日 ≤ 終了日 | 「開始日は終了日以前を指定してください」 |

### (3) アクションバー

**要素構成図:**

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ (3) アクションバー                                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                     (3.1) [CSVエクスポート]  (3.2) [+ 台帳登録]  (3.3) [削除]│
└─────────────────────────────────────────────────────────────────────────────┘
```

**3-1: CSVエクスポートボタン**
- 概要: 現在の検索条件でフィルタリングされたデータをCSV出力
- スタイル: セカンダリボタン（白背景、青枠）
- 有効条件: 常時有効
- ファイル名形式: `device_inventory_{YYYYMMDD_HHmmss}.csv`

**3-2: 台帳登録ボタン**
- 概要: 新規台帳登録モーダルを開く
- スタイル: プライマリボタン（青背景）
- 有効条件: 常時有効

**3-3: 削除ボタン**
- 概要: 選択された台帳を削除
- スタイル: 危険ボタン（赤背景）
- 有効条件: チェックボックスが1件以上選択されているとき
- 無効時: グレーアウト表示
