## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | `ALT-001` |
| 画面名 | アラート設定一覧画面 |
| 画面パス | `/alert/alert-setting` |
| アクセス権限 | システム保守者、管理者、販社ユーザ、サービス利用者 |
| 親画面 | [ダッシュボード](../dashboard/ui-specification.md) |

### レイアウト構成

| No | 領域 | 説明 | 備考 |
|------|------|------|------|
| (1) | ヘッダー | アイコン、画面タイトル、アクションボタン | 固定表示 |
| (2) | 検索フォーム | 検索条件入力エリア | 折りたたみ可能 |
| (3) | データテーブル | 検索結果の一覧表示（ページネーション含む） | スクロール可能 |
| (4) | アラート設定登録モーダル | 登録フォーム | モーダルダイアログ |
| (5) | アラート設定更新モーダル | 更新フォーム | モーダルダイアログ |
| (6) | アラート設定参照モーダル | 詳細情報表示 | モーダルダイアログ（読み取り専用） |
| (7) | アラート設定登録確認モーダル | 登録情報確認 | モーダルダイアログ（読み取り専用） |
| (8) | アラート設定更新確認モーダル | 更新情報確認 | モーダルダイアログ（読み取り専用） |
| (9) | アラート設定削除確認モーダル | 削除確認 | モーダルダイアログ（読み取り専用） |
| (10) | アラート設定登録完了モーダル | 登録完了通知 | モーダルダイアログ（読み取り専用） |
| (11) | アラート設定更新完了モーダル | 更新完了通知 | モーダルダイアログ（読み取り専用） |
| (12) | アラート設定削除完了モーダル | 削除完了通知 | モーダルダイアログ（読み取り専用） |
