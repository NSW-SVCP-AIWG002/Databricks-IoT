## 📝 開発メモ

### 注意事項
1. **必ずBunを使用**: npm/yarnは使用禁止（CLAUDE.md参照）
2. **appTheme使用必須**: 直接的な色指定・サイズ指定は禁止
3. **Conventional Commits準拠**: コミットメッセージ規約を遵守
4. **TypeScript厳格モード**: anyの使用は極力避ける
