# ローカルルール・開発規約

このディレクトリには、Databricks IoTシステムプロジェクトの開発における各種ルールや規約が格納されています。

## 📋 ドキュメント一覧

### general-rule.md

- 日付の取り扱いルール

### flask-jinja2-rule.md

- View関数とテンプレートの責務分離
- ビジネスロジックとデータ取得の分離
- テンプレート構造（継承、マクロ、インクルード）

### css-design-token-rule.md
CSSデザイントークン管理のルール

- 直接値指定の禁止

### test-rule.md
テスト実装のルール
- テスト実装の基本原則（AAA パターン、独立性等）

- テスト実行コマンド一覧
- テストコードのファイル配置ルール
